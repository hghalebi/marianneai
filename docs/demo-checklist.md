# Demo Checklist

- Confirm the shared request and response contract is current.
- Confirm the source catalog only lists official French public data sources.
- Test one inbound email from receipt to reply.
- Test one web submission from form input to rendered answer.
- Verify both flows call the same open-data service.
- Prepare at least two demo scenarios from `shared/demo-scenarios`.
- Rehearse one housing scenario, one air-quality scenario, or one European-funds scenario.
- Confirm the human review step is visible before send.
- Record the demo video after the end-to-end test passes.
- Submit only after the email and web paths both succeed.
