# Shared Prompts

Use this directory for prompt assets, retrieval instructions, and reusable generation templates shared across services.

Keep prompt changes here when they influence both the email and web flows through the open-data service.

## Guardrails

- Use only official French public data sources.
- Produce clear language for citizens and public agents.
- Cite every source used in the answer.
- State uncertainty when the underlying data is incomplete.
- Preserve a reviewable draft rather than pretending the answer is final.
