# Shared Data Sources

Use this directory to track the official French public data sources approved for retrieval and citation.

## Priority Sources

- `data.gouv.fr`
- `INSEE`
- `OpenDataSoft` portals from French collectivites
- `DREES`
- `CEREMA`
- `Etalab`
- Relevant official APIs and territorial open-data portals

## Rules

- Prefer official French public sources over secondary summaries.
- Keep source names consistent so citations look the same in email and web.
- Record retrieval dates when a source is time-sensitive.
- Add demo-relevant datasets here before wiring them into the open-data service.
